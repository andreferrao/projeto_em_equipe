# Restaurante Sabor & Arte

## Descrição
Site institucional para o Restaurante Sabor & Arte - Um projeto simples de HTML com Tailwind CSS para demonstração de Git e GitHub.

## Estrutura do Projeto

```
terceiro_remoto/
│
├── index.html              # Página inicial
├── README.md               # Documentação do projeto
│
├── pages/                  # Páginas internas
│   ├── menu.html          # Cardápio do restaurante
│   ├── galeria.html       # Galeria de fotos
│   └── contato.html       # Página de contato
│
├── assets/                 # Recursos estáticos
│   └── images/            # Imagens
│       ├── pratos/        # Fotos das refeições
│       └── ambiente/      # Fotos do interior do restaurante
│
└── css/                    # Estilos personalizados
    └── styles.css         # CSS adicional

```

## Tecnologias Utilizadas

- HTML5
- Tailwind CSS (via CDN)
- CSS3 (para customizações)

## Como Visualizar

1. Clone o repositório
2. Abra o arquivo `index.html` em um navegador
3. Navegue pelas páginas através do menu

## Páginas

- **Início**: Apresentação do restaurante e pratos em destaque
- **Menu**: Cardápio completo com entradas, pratos principais e sobremesas
- **Galeria**: Fotos dos pratos e do ambiente
- **Contato**: Formulário de contato e informações

## Notas para o Curso

Este projeto foi criado especificamente para demonstrar:
- Estrutura de pastas organizada
- Versionamento com Git
- Trabalho colaborativo no GitHub
- Boas práticas de organização de código
